\begin{frame}{Project Overview}
\begin{block}{What this project is}
This project studies \textbf{fairness-aware routing under missing context} in two environment families: \textbf{clinical decision support} and \textbf{quantum-network routing}.
\end{block}
\begin{block}{Shared problem}
Both domains are \textbf{sequential decision systems}: they allocate scarce resources under uncertainty, observe only partial feedback, and update future decisions from incomplete evidence.
\end{block}
\textbf{Project goals}
\begin{itemize}
    \item Compare \mab, \cmab, and \icmab{} strategies.
    \item Measure both \textbf{performance} and \textbf{fairness}.
    \item Show how missing or degraded context can create repeated allocation harm.
    \item Build a reproducible framework with reportable artifacts.
\end{itemize}
\vfill
\slidefoot{Acronyms: SDS = sequential decision system; MAB = multi-armed bandit; CMAB = contextual multi-armed bandit; iCMAB = informative contextual multi-armed bandit. Sources: Auer et al. (2002); Li et al. (2010); Lattimore and Szepesvari (2020); DSCI601 Project Proposal.}
\end{frame}



\begin{frame}{Why This Project Matters}
\begin{columns}[T]
\begin{column}{0.48\textwidth}
\vspace{-0.5cm}
\begin{block}{Clinical impact}
Routing decisions affect real patient outcomes.
\end{block}

\begin{itemize}
    \item Who receives a diagnostic test?
    \item Who gets retested or escalated?
    \item Whose risk is missed because context is incomplete?
\end{itemize}

\alert{\scriptsize A policy can look accurate on average while still delaying access to critical resources/care for a vulnerable group.}
\end{column}

\begin{column}{0.48\textwidth}
\vspace{-0.5cm}
\begin{block}{Quantum-network impact}
Routing decisions affect service quality under noisy conditions.
\end{block}

\begin{itemize}
    \item Which path gets selected?
    \item Which flows receive scarce qubit or link resources?
    \item Which users experience repeated failure or latency?
\end{itemize}

% \vspace{0.4em}

\alert{\scriptsize A policy can maximize total throughput while still giving some flows consistently worse service.}
\end{column}
\end{columns}

% \vspace{0.6em}

\begin{block}{Core reason this matters}
Fairness-aware routing asks whether the system is making good decisions
\textbf{for everyone affected}, not only good decisions on average.
\end{block}

\vfill
\slidefoot{Acronyms: none introduced on this slide. Clinical sources: Sjoding et al. (2020); Fawzy et al. (2022). Quantum-network sources: Wehner et al. (2018); Pant et al. (2019); DSCI601 quantum-routing materials.}
\end{frame}

\end{block}

    \item service weights: emergency, inpatient, outpatient

\end{column}

\begin{column}{0.30\textwidth}
\textbf{\textcolor{phase4purple}{Mediation boundary}}
\begin{itemize}
    \item \texttt{ClinicalEquitasMediator}
    \item \texttt{QuantumEquitasMediator}

    \item modes: audit vs. mitigate
    \item emits priority, success, latency, and reward adjustments
\end{itemize}
\end{column}
\end{columns}

\vspace{-0.1cm}

\begin{block}{Current status}
This layer is wired into execution: each clinical case produces decisions such as a model, an EQUITAS mediation, an allocation decision, and observed outcomes. Those fields are stored in the run state, so we can trace harm back to the policy action, mediation mode, priority score, allocation label, and partial-feedback outcome.
\end{block}

    \item \textbf{Scale the clinical evidence cases:} expand test allocation and reliability scenarios across seeds, cohorts, missingness levels, and policy families.
    \item \textbf{Deepen fairness reporting:} move beyond artifact existence and model summaries into stronger disparity analysis, uncertainty, effect size, and cross-setting comparisons.
    \item \textbf{Close the feedback loop:} use EQUITAS/fairness evidence to update future routing decisions.
    \item \textbf{Finish the validation hub:} link every final research claim to run specs, generated states, figures, manifests, source evidence, and reproducibility metadata.
\end{enumerate}
